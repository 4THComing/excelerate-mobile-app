# Excelerate Mobile App

Cross-platform mobile application developed using Flutter for the Excelerate Internship Program.

## Overview

The Excelerate Mobile App is designed to help learners discover and engage with educational programs through a simple and user-friendly mobile interface. This project was developed as part of the Excelerate Internship Program and focuses on applying Flutter fundamentals such as UI design, navigation, layouts, and application structure.

## Features

* Login Screen
* Home Screen
* Program Listing Screen
* Program Details Screen
* Navigation using Named Routes
* Global ThemeData Configuration
* Responsive Flutter UI Components

## Screens Implemented

### Login Screen

Allows users to enter their email and password and navigate into the application.

### Home Screen

Acts as the central hub of the application and provides access to key sections such as Programs, Announcements, and Quick Links.

### Program Listing Screen

Displays available programs for learners to browse.

### Program Details Screen

Provides detailed information about a selected program and includes a registration button.
